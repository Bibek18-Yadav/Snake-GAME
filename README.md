# Snake-GAME
This is a beginner friendly Snake Game built with Python and Pygame. It helped me understand game loops, keyboard controls, and basic logic.

Built With:
 1. Python 3
 2. Pygame
 3. Random module
 4. Keyboard input(arrow keys)

How to Run Locally:
  1. Install Python 3:
      https://www.python.org/downloads/
  2. Install Pygame:
      pip install pygame
  3. Clone or Download the Repo:
      git clone https://github.com/Bibek18-Yadav/Snake-GAME.git
  4. Run the Game:
      python snake_game.py

Controls:

  1. Arrow Keys → Move the snake
  2. P → Play again
  3. Q → Quit game

Notes:
  1. Code is intentionally simple and beginner-friendly
  2. Made mainly for practice and learning

Happy coding.....
